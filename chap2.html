Chapter 2: Styling your website

Now that you've got a basic website, you'll want to make it more appealing.

The language that's used to add style to websites is called "CSS", nowadays we also use different languages for styling but you'll want to learn the basics first.

  • (Create a folder for your website, during the course of these tutorials you'll need to add more files to your website)
  • Learn the CSS basics
  • Tip: CSS is included in the HTML file, the tutorial will walk you through. For future ease always use the "External Stylesheet" method. It allows you to include the same style in more than one HTML file. (This makes editing a large website easier)
